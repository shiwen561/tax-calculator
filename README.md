# 税务计算助手

专为会计人员打造的中国全税种计算器，支持个人所得税、增值税、企业所得税、房产税、印花税等多种税种计算。

## 🚀 功能特性

- ✅ **多税种支持** - 个人所得税、增值税、企业所得税、房产税、印花税、契税、土地增值税等
- ✅ **互动式计算** - 分步式输入，实时显示结果和明细
- ✅ **PWA 支持** - 可在安卓手机上像原生 App 一样安装使用
- ✅ **历史记录** - 自动保存计算记录，支持按税种筛选
- ✅ **常用主体** - 预设企业常用税费计算模板
- ✅ **导出功能** - 支持导出 Excel 和 HTML 报告

## 📱 安卓手机安装使用

### 方法一：使用 Vercel 云端地址（推荐）

1. 在安卓手机浏览器中打开：
   ```
   https://tax-calculator-jhy59j9ji-shiwenhuang891-3718s-projects.vercel.app
   ```
2. 浏览器会提示"添加到主屏幕"或弹出安装提示
3. 点击安装，即可在手机桌面看到"税务计算器"图标

### 方法二：本地部署

1. 克隆项目
2. 安装依赖：`npm install`
3. 构建项目：`npm run build`
4. 将 `dist/` 文件夹部署到任何静态托管服务

## 💻 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📝 税种列表

1. 个人所得税（工资薪金、全年奖、劳务报酬、稿酬、特许权使用费）

3. 企业所得税（小型微利企业、一般企业）
4. 房产税
5. 印花税
6. 城建税及附加
7. 土地增值税
8. 契税
