                        税务计算助手 - GitHub 上传指南
================================================================================

方法一：使用 GitHub 网页上传（最简单）
----------------------------------------

1. 打开浏览器，访问：https://github.com/shiwen561/-

2. 点击页面中间的 "uploading an existing file"（如果仓库是空的）
   或者点击 "Add file" → "Upload files"

3. 将以下所有文件和文件夹拖放到上传区域：
   - src/ 文件夹（整个文件夹）
   - public/ 文件夹（整个文件夹）
   - index.html
   - package.json
   - package-lock.json
   - vite.config.ts
   - tsconfig.json
   - tsconfig.node.json
   - tailwind.config.js
   - postcss.config.js
   - README.md
   - .gitignore

   注意：不需要上传 node_modules/ 文件夹！

4. 在底部 "Commit changes" 框中输入：
   标题："初始提交：税务计算助手"

5. 点击绿色的 "Commit changes" 按钮！

完成！现在您的项目已经在 GitHub 上了！

================================================================================

方法二：下载使用
-----------------

1. 项目上传到 GitHub 后，访问 https://github.com/shiwen561/-

2. 点击绿色的 "Code" 按钮

3. 选择 "Download ZIP" 下载项目压缩包

4. 解压后，使用方式见 README.md

================================================================================

方法三：直接使用已部署的版本
----------------------------

您现在就可以直接使用，不需要下载！

- 在线访问：https://tax-calculator-jhy59j9ji-shiwenhuang891-3718s-projects.vercel.app

- 安卓手机安装：
  1. 用手机浏览器打开上面的地址
  2. 点击浏览器菜单 → "添加到主屏幕"
  3. 即可像 App 一样使用！
